# SECURE-PAYMENT-INTEGRATION

## Overview

ASP.NET Core MVC application with Stripe payment gateway integration.

## Features

- Secure Card Payments
- Stripe Checkout
- Payment Success Page
- Payment Cancel Page

## Technologies

- ASP.NET Core MVC
- Stripe API
- .NET 8

## Installation

```bash
dotnet restore
dotnet run
```

## Open

https://localhost:5001
